**RobloxTools** is an advanced, commercial-grade automated tool designed for Roblox players who need their game instances to run 24/7 without interruptions. Built from the ground up for massive farms, it features a smart log-tracking system, a multi-account manager, hardware-level security, and a built-in Telegram Bot for remote control.

---

## ✨ Features (English)

* **📱 Telegram Remote Control (NEW):** Control your entire farm from your phone! Send commands to your personal Telegram bot to get live screenshots (`/screen`), check stats (`/status`), restart specific accounts (`/restart Name`), change games (`/setplace`), or even reboot your PC (`/reboot`).
* **👥 Multi-Account Manager:** Run and manage a massive farm with perfect Process ID (PID) isolation. If one bot out of 10 crashes, *only* that specific instance will be restarted. The rest of your farm remains completely untouched!
* **🔥 Smart Log Scanner:** The engine actively monitors system logs in real-time. It instantly detects game freezes, kicks, timeouts, and disconnections (including Errors 277, 268, 279, 290, 273, and 267).
* **🔒 Ultimate Security & Privacy:** License keys are strictly hardware-locked (HWID) to your PC. Keys are hidden by default (`••••••••`) on startup to protect streamers. Local session data is protected with banking-grade DPAPI encryption.
* **👻 Ghost Mode:** Completely hide active game windows from your desktop and taskbar to save PC resources and keep your workspace clean.
* **🌐 Dynamic Localization:** Instantly switch the UI, console logs, and Telegram bot messages between English and Russian without restarting the app.
* **🔔 Live Webhooks:** Receive real-time status reports, restart alerts, and crash screenshots directly to your **Discord** or **Telegram**.

### 📥 Getting Started
1. Go to the [Releases](../../releases) tab and download the latest **`RobloxTools.exe`** file.
2. Place the file in a **new empty folder** on your PC.
3. Run `RobloxTools.exe` and enter your License Key.
4. **(Optional)** Open **Settings -> Webhooks** and paste your Telegram Bot Token & Chat ID to enable remote control.
5. Add your accounts via the **Accounts** menu, input your Place ID, and click **INITIATE SYSTEM**.

### 📥 Установка и Запуск
1. Перейдите во вкладку [Releases](../../releases) и скачайте актуальный файл **`RobloxTools.exe`**.
2. Поместите файл в **новую пустую папку** на компьютере.
3. Запустите `RobloxTools.exe` и введите ваш ключ активации.
4. **(Опционально)** Зайдите в **Настройки -> Webhooks** и вставьте Token и Chat ID вашего бота для включения удаленного управления.
5. Добавьте аккаунты в меню **Аккаунты**, укажите Place ID и нажмите **ЗАПУСТИТЬ**.

## 🤖 Telegram Bot Commands / Команды бота
Once configured, send `/help` to your bot to see all commands:
* **`/status`** - System stats & uptime / Инфо о ферме
* **`/screen`** - Request a live PC screenshot / Запросить скриншот ПК
* **`/start` & `/stop`** - Toggle monitoring / Вкл/выкл мониторинг
* **`/restart [Name]`** - Force restart a specific bot / Перезапуск конкретного окна
* **`/setplace [ID]`** - Change the target game / Сменить игру для фарма
* **`/killall` & `/restart_all`** - Global process control / Глобальное управление окнами
* **`/reboot` & `/shutdown`** - Remote PC power management / Управление питанием ПК
